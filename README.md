# vue和nodejs聊天室
看到了个不错的聊天室代码分享，原作者代码为一个页面有五个不同chatroom代表五个人，并实现了ws通信。  
在基础上做了一些调整修改，每个人登录页面可以选择角色（模拟登录）  
将对话框左右分割区分更好的区分主副人物消息，添加了时间和系统消息等小demo，新人员进入后触发系统提示
